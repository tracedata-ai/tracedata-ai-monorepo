# Module 1 XRAI Compliance Skill

Our project is TraceData.ai. For every feature we build, we must satisfy the IMDA Model AI Governance Framework.

## Guidelines

1. **Explainability First**: If a feature involves AI or decision making, check for explainability components (XAI).
2. **Warn Immediately**: If XAI is missing from a feature spec or implementation, warn me immediately.
3. **Integration**: Suggest integration of SHAP or LIME where appropriate for model interpretability.
4. **Transparency**: Ensure model decisions are transparent and logged for audit purposes.
5. **Governance**: Verify compliance with IMDA Model AI Governance Framework principles.
