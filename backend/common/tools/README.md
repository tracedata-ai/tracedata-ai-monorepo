# common/tools

The Tool Registry. Shared modules imported by agents to perform specific actions.

- **context/**: Weather, Maps, Road Incidents, Traffic tools.
- **ml/**: XGBoost, SHAP, and Fairness evaluation tools.
- **llm/**: Standardized LLM call wrappers.
- **data/**: Vector search, embeddings, and notification tools.
- **emergency/**: SCDF and LTA dispatch clients and tools.
