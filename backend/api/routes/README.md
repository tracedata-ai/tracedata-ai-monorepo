# api/routes

FastAPI route handlers. Part of the main API container.

- **trips.py**: Endpoints for trip listing and details.
- **scores.py**: Endpoints for behavior and smoothness scores.
- **explain.py**: Endpoints for SHAP/XAI retrieval.
- **coaching.py**: Endpoints for driver coaching logs.
- **feedback.py**: Endpoints for driver feedback and disputes.
- **health.py**: Liveness probes and queue depth monitoring.
